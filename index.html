Example

About

Created by Lauren Yoo, who developed this after constantly looking at their LeetCode submission tracker during their job hunt and, for some reason, really liked how this particular heatmap looked. bathtiles.js was named after the retro bathroom tiles that the heatmap looks similar to, specifically the mosaic kind.

Made with:

  • d3
  • Tippy.js

Installation

Run npm install bathtiles.js in the terminal.

Usage

        
            const target = document.getElementById('heatmap');

            const bathtiles = new Bathtiles();

            bathtiles.load(target);
